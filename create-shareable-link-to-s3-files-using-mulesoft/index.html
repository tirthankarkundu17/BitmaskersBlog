Create Shareable link to private S3 Files using MuleSoft

Oct 29, 2021 2 min read mulesoftaws

AWS is one of the most popular Cloud platform. Check a detailed post on AWS S3 here:

The Problem

Now let us talk about the use case we are trying to solve.

We want to share some of the private files in AWS S3 to external users for a limited period.

The Solution

AWS S3 allows us to solve this problem using Presigned URLs.

All objects by default are private. Only the object owner has permission to access these objects. However, the object owner can optionally share objects with others by creating a presigned URL, using their own security credentials, to grant time-limited permission to download the objects.

Now let us create the integration using MuleSoft

Create a private flow or a sub flow to generate presigned url

  3. We set the time till which the url is suppose to be active
  4. We use the S3 Create object presigned uri operation to generate shareable url
