Step 1: Flash Raspberry Pi OS
  • Download Raspberry Pi Imager
  • Flash Raspberry Pi OS (64-bit) or the Lite version to your SD card
  • Add a hostname like pihole1.local and fill the WIFI Config
  • Enable SSH
  • Let the imager write the OS to SD card
  • Insert the SD card and boot up your Pi
Step 2: Connect & Update

SSH into your Pi or use a monitor/keyboard. Then:

Run below command

sudo apt update && sudo apt upgrade -y

Step 3: Set a static IP for your Pi

We need to set a static IP for the pi since pi hole will work as a server.

We have two ways to do this -

  1. Use your router to setup a static IP
  2. Use the pi itself 
sudo vi -w /etc/dhcpcd.conf

And then scroll down until you find the section called ‚ÄòExample Static IP configuration:‚Äô

This section is commented out by default, so you will need to uncomment some lines and input your own static IP address and network information.

In my case the configuration will look like this:

# Example static IP configuration:
interface wlan0
static ip_address=192.168.26.145/24
static routers=192.168.26.1
static domain_name_servers=192.168.26.1 1.1.1.1

I am using my WIFI hence I set the interface as wlan0, if you are using ethernet then you may want to set interface to eth0.

Step 4: Install Pi-hole

Installing pi hole is as easy as running - 

curl -sSL https://install.pi-hole.net | bash

Now you need to follow the onscreen commands -

Since we already set the Static IP in Step 3, we can click on Continue and proceed

I am using WIFI so I will select wlan0, if you are using ethernet - use eth0

I am using Cloudflare as my DNS provider here you can select Google or others too.

All right, our installation is complete.
